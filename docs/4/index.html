Conditional Breakpoints

Conditional breakpoints are very useful when there is a loop but you only want the program to stop when a certain value is reached. You can create a conditional breakpoint like a normal one, except you right click on the red circle and choose “Add Conditional Breakpoint …”.

Or if you right click on a breakpoint you’ve made and choose “Edit Breakpoint…”, you can make it a conditional one.

Now you are being prompted a ‘Expression’ to determine when it breaks. You can write any Javascript expressions, even make function calls. Just make sure to hit Enter after setting your expression to save, or Esc to cancel.

Summary

We have the power to set the breakpoints to only activate when we want them to. This can save us a lot of time when we only want the program to stop when a specific value is reached rather than constantly step until we get that value.
